# Riiyazii Music Academy

Official website for Riiyazii Music Academy, led by Dr. Nikunj Gautham. Located in Nikol, Ahmedabad, Gujarat.

## Features
- **Modern Design**: High-end, responsive UI with glassmorphism effects.
- **SEO Optimized**: Optimized for Local SEO, EO, and AEO targeting Nikol and Ahmedabad.
- **Instrument Specialization**: Optimized for Guitar, Piano, Harmonium, Sitar, and Tabla classes.
- **Interactive**: Scroll animations and video backgrounds.

## Tech Stack
- HTML5 / Vanilla CSS
- JavaScript (Vite)
- Schema.org (JSON-LD) for Local Business and FAQ

## Deployment
This project is ready for deployment on [Vercel](https://vercel.com).
Simply push to GitHub and connect the repository.

## Local Development
1. Install dependencies: `npm install`
2. Start dev server: `npm run dev`
3. Build for production: `npm run build`
